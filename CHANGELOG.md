# Changelog
所有显著的变更都会记录在这个文件中，遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.0.0/) 规范。
本项目的版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/)（主版本.次版本.补丁版本，如 1.2.3）。

#### 2026-6-27 (1.3-beta.3)

- **远程容器物品获取与智能回塞系统** — 通过网络数据包远程扫描/请求容器物品，自动取料建建，FIFO 回塞，缓存持久化
- **惰性扫描系统** — BlockUpdate 脏坐标追踪 + LAZY/PARTIAL/FULL 三态状态机，空闲自动降级减少性能开销
- **高亮渲染系统** — 可配置的方块放置预览高亮
- **重写 QuickShulkerUtils** — 支持 MOD/PLUGIN 双模式，适配插件服
- **俄语 (ru_ru) 本地化**（感谢 Rofumer）
- **支持 MC 26.2**
- **性能优化** — 降低 tick 迭代开销，修复空背包 FPS 骤降，在 box 层级跳过图层过滤方块，缓存 eyePos（感谢 Rofumer）
- **重构与清理** — QuickShulker 双模式、移除遗留 ChestTracker/Jackfred/OpenInv 集成代码（-4700+ 行）、清理构建脚本与未使用依赖、更新命名规范
- **修复** — 填充/流体空跑、手长判定不一致/边界方块丢失、破冰放水死循环、含水方块水位判断、Mixin 崩溃、远程容器缓存路径与同步问题、轻松放置不从潜影盒取物、染色玻璃板误判
